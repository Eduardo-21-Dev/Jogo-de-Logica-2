Licao 6: fase final - troca cruzada dos blocos

Desafio final

Tres zonas: A, C e B com labirintos diferentes

Objetivo desta fase: realizar uma troca completa de blocos entre tres zonas separadas. Leve o bloco da Area A para a base C, o bloco da Area C para a base B e o bloco da Area B para a base A.

Regras da fase final:

  • 1. Use pegar() para carregar um bloco e largar() para soltar
  • 2. Existe uma passagem A->C que teleporta o jogador para a Zona C
  • 3. A porta C->B inicia fechada e abre somente apos A->C
  • 4. A porta B->A inicia fechada e abre somente apos C->B
  • 5. A base A fica isolada ate a porta B->A abrir
  • 6. A fase termina apenas com os 3 blocos nas bases corretas

Comandos do jogador

Tres zonas separadas: use o teleporte A->C e conclua A->C, C->B e B->A.

Objetivo: concluir a troca A->C, C->B e B->A para vencer a fase final.

Portas: C->B FECHADA | B->A FECHADA. Objetivos concluidos: 0/3

  • Levar bloco A para base C
  • Levar bloco C para base B
  • Levar bloco B para base A

Comando invalido

Funcoes existentes

  • moverDireita
  • moverEsquerda
  • moverCima
  • moverBaixo
  • Com parametro: moverDireita(3)
  • pegar()
  • largar()

A passagem A->C teleporta o jogador para a Zona C. Cadeia de portas: A->C abre C->B e C->B abre B->A.

Area do labirinto final

Leve os blocos para bases de troca e destranque a porta C->B no meio da partida.
